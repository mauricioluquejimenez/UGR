Práctica 2. Pilas y Colas
TDAs lineales: Pilas y colas con máximo

Introducción

En esta práctica se representan dos TDA, la pila y la cola, implementados mediante el uso de colas y pilas respectivamente. Para ello, se hace uso de contenedores auxiliares para simular el funcionamiento de cada TDA. Además, cada TDA permite almacenar el valor máximo existente en cada momento. Para cumplir estos dos objetivos, se han sobreescrito las funciones básicas de una pila y una cola, tales como devolver un elemento, borrarlo, comprobar si la estructura está vacía, y, principalmente, insertar un nuevo valor. Es aquí donde se hace la comprobación y la actualización del valor máximo de la estructura, en el momento de la inserción de un valor.
