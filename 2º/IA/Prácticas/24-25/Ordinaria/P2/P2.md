# 5.Resolviendo las interacciones entre los agentes

Básicamente, el procedimiento que se ha llevado a cabo para evitar la colisión con otros agentes es el siguiente:

- Sólo el agente rescatador evita colisiones. El auxiliar, por tanto, simplemente se mueve como si nada.

- Lo primero que comprueba el agente rescatador es si justo delante de él hay otro agente, en este caso un auxiliar. Esta comprobación se realiza junto con la comprobación relativa a si está en un puesto base, ya que la reacción es la misma: no hacer nada.
- Si se encuentra con un agente (en este caso un auxiliar) fuera de una casilla de puesto base, sí se requiere un movimiento, por lo que se realiza un giro de 90º a la derecha, o lo que es lo mismo, 2 giros de 45º.

- Se incluye una última comprobación para comprobar si, en vez de justo delante, el agente auxiliar se encuentra en alguna otra casilla adyacente. En este caso, el giro a realizar (también hacia la derecha) es únicamente de 45º.
